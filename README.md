# 🦠 Global Disease Outbreak Tracker

https://disease-tracker.streamlit.app/

**Disease Tracker** - это аналитический веб-дашборд, который в реальном времени отслеживает мировые новостные сводки, использует ИИ для фильтрации данных и визуализирует потенциальные эпидемиологические угрозы на карте.

## Ключевые возможности

* **Real-time Data Pipeline:** Автоматический сбор свежих данных через NewsAPI с умной фильтрацией по ключевым паттернам болезней.
* **AI NLP Filtering:** Использование Zero-Shot Classification (Hugging Face) для анализа контекста заголовков и отсева нерелевантных новостей (например, исторических справок или обсуждений вакцин).
* **Geo-Spatial Analysis:** Извлечение гео-сущностей с помощью `spaCy` и геокодирование через `GeoPy`.
* **User-Centric Location:** Запрос точных GPS-координат пользователя через браузер с автоматическим расчетом дистанции до ближайшего очага заболевания.
* **Interactive Mapping:** Динамическая карта на базе `Folium` с отображением очагов и текущей позиции пользователя.

## Используемый стек

* **Деплой:** Streamlit, Streamlit Community Cloud
* **Скрипт:** Python
* **NLP:** Transformers (Hugging Face), spaCy
* **Гео-аналитика:** Folium, GeoPy, Haversine formula
* **Работа с данными:** Requests, JSON
* **Базы данных / Кэширование:** SQLite, SQL
